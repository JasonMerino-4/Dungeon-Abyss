# Dungeon-Abyss
**Date:** November 2023
**Credits:** Starter code provided by Scott Reckinger @UIC

Summary: Dungeon Abyss is a terminal game where you play as a hero in a dungeon filled with dangers. You're two options as a player are to either outlast the bats and monsters or to make it to the escape.

How to play:
  awsd controls: a (left), w (up), s (down), d(right)
  
  #'s: Are used to represent holes, avoid moving your hero into these or it's game over. You can also lead monsters into these holes causing them to dissapear from the board
  +'s: Are used to represent barriers you can't move on spaced occupied by these and neither can the monsters
  M's: Are use to represent big monsters capables of moving to units at a time
  m's: Are used to represent smalls monsters capables of moving only one unit at a time
  ~'s: Are used to represent bats, these can only move horizontally
  H: is used to represent you the player and Hero
  *: is used to represent the exit
